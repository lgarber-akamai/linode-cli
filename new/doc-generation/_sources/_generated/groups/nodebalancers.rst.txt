configs



.. rst-class:: action-argument-section-table

..  list-table::
    :header-rows: 1

    * - Name
      - Required
      - Type
      - Description

    * - :action-table-field-name:`\-\-configs.algorithm` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`string`
      - What algorithm this NodeBalancer should use for routing traffic to backends.

    * - :action-table-field-name:`\-\-configs.check` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`string`
      - The type of check to perform against backends to ensure they are serving requests.

    * - :action-table-field-name:`\-\-configs.check_attempts` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`integer`
      - How many times to attempt a check before considering a backend to be down.

    * - :action-table-field-name:`\-\-configs.check_body` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`string`
      - This value must be present in the response body of the check in order for it to pass.

    * - :action-table-field-name:`\-\-configs.check_interval` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`integer`
      - How often, in seconds, to check that backends are up and serving requests.

    * - :action-table-field-name:`\-\-configs.check_passive` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`boolean`
      - If true, any response from this backend with a :code:`5xx` status code will be enough for it to be considered unhealthy and taken out of rotation.

    * - :action-table-field-name:`\-\-configs.check_path` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`string`
      - The URL path to check on each backend.

    * - :action-table-field-name:`\-\-configs.check_timeout` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`integer`
      - How long, in seconds, to wait for a check attempt before considering it failed.

    * - :action-table-field-name:`\-\-configs.cipher_suite` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`string`
      - What ciphers to use for SSL connections served by this NodeBalancer.

    * - :action-table-field-name:`\-\-configs.nodes` :ref:`(section) <commands_nodebalancers_create_argument_sections_configs.nodes>`
      - :action-table-field-optional:`No`
      - :action-table-field-type:`json`
      - A NodeBalancerNode represents a single backend serving requests for a single port of a NodeBalancer.

    * - :action-table-field-name:`\-\-configs.port` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`integer`
      - The port this Config is for.

    * - :action-table-field-name:`\-\-configs.protocol` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`string`
      - The protocol this port is configured to serve.

    * - :action-table-field-name:`\-\-configs.proxy_protocol` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`string`
      - ProxyProtocol is a TCP extension that sends initial TCP connection information such as source/destination IPs and ports to backend devices.

    * - :action-table-field-name:`\-\-configs.ssl_cert` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`string`
      - The PEM-formatted public SSL certificate (or the combined PEM-formatted SSL certificate and Certificate Authority chain) that should be served on this NodeBalancerConfig's port.

    * - :action-table-field-name:`\-\-configs.ssl_key` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`string`
      - The PEM-formatted private key for the SSL certificate set in the :code:`ssl_cert` field.

    * - :action-table-field-name:`\-\-configs.stickiness` 
      - :action-table-field-optional:`No`
      - :action-table-field-type:`string`
      - Controls how session stickiness is handled on this port.
