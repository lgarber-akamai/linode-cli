.. _commands_placement_assign-linode:

`assign-linode <https://techdocs.akamai.com/linode-api/reference/post-group-add-linode>`_
-----------------------------------------------------------------------------------------


Add one or more compute instances to your placement group.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli placement assign-linode [-h] [--linodes linodes]
    	groupId


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli placement assignLinode 528 \
    	--linodes 123 456 \
    	--non-compliant true

.. _commands_placement_group-create:

`group-create <https://techdocs.akamai.com/linode-api/reference/post-placement-group>`_
---------------------------------------------------------------------------------------


Creates a placement group on your account.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli placement group-create [-h] [--region region]
    	[--placement_group_policy placement_group_policy]
    	[--placement_group_type placement_group_type] [--label label]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli placement create \
    	--label PG_Miami_failover \
    	--region us-mia \
    	--placement_group_type "anti-affinity:local" \
    	--placement_group_policy strict


.. cssclass:: action-subheading

    .. rubric:: Arguments

.. cssclass:: action-subheading-description

    Additional fields used to execute this request.


.. rst-class:: action-argument-section-table

..  list-table::
    :header-rows: 1
    :width: 100%
    :widths: 1 10 10 79

    * - Name
      - Required
      - Type
      - Description

    * - :code:`--label`
      - :action-argument-required:`Yes`
      - string
      - A unique name for the placement group.
    * - :code:`--placement_group_policy`
      - :action-argument-required:`Yes`
      - string
      - How requests to add future compute instances to your placement group are handled:  - :code:`strict`.
    * - :code:`--placement_group_type`
      - :action-argument-required:`Yes`
      - string
      - How compute instances are distributed in your placement group.
    * - :code:`--region`
      - :action-argument-required:`Yes`
      - string
      - The data center that houses the compute instances you want to add to your placement group.

.. _commands_placement_groups-list:

`groups-list <https://techdocs.akamai.com/linode-api/reference/get-placement-groups>`_
--------------------------------------------------------------------------------------


Returns a paginated list of placement groups you have permission to view.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli placement groups-list [-h]
    	[--placement_group_type placement_group_type]
    	[--region region] [--label label]
    	[--order-by {placement_group_type,region,label}]
    	[--order {asc,desc}]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli placement-list


.. cssclass:: action-subheading

    .. rubric:: Filterable Attributes


.. cssclass:: action-subheading-description

    Arguments used to define a filter for response entries.


.. rst-class:: action-filterable-field-table

..  list-table::
    :header-rows: 1
    :width: 100%
    :widths: 15 10 75

    * - Name
      - Type
      - Description

    * - :code:`--label`
      - *string*
      - The unique name set for the placement group.
    * - :code:`--placement_group_type`
      - *string*
      - How compute instances are distributed in your placement group.
    * - :code:`--region`
      - *string*
      - The `region <https://techdocs.akamai.com/linode-api/reference/get-regions>`_ where the placement group was deployed.
