.. role:: action-argument-required

Stackscripts
============

This section details Stackscript-related Linode CLI commands.

.. _commands_stackscripts_list:

`list <https://techdocs.akamai.com/linode-api/reference/get-stack-scripts>`_
----------------------------------------------------------------------------


If the request is not authenticated, only public StackScripts are returned.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli stackscripts list [-h] [--label label]
    	[--description description]
    	[--deployments_total deployments_total] [--rev_note rev_note]
    	[--is_public is_public] [--mine mine]
    	[--order-by {label,description,deployments_total,rev_note,is_public,mine}]
    	[--order {asc,desc}]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli stackscripts list


.. cssclass:: action-subheading

    .. rubric:: Filterable Attributes


.. cssclass:: action-subheading-description

    Arguments used to define a filter for response entries.


.. rst-class:: action-filterable-field-table

..  list-table::
    :header-rows: 1
    :width: 100%
    :widths: 15 10 75

    * - Name
      - Type
      - Description

    * - :code:`--deployments_total`
      - *integer*
      - The total number of times this StackScript has been deployed.
    * - :code:`--description`
      - *string*
      - A description for the StackScript.
    * - :code:`--is_public`
      - *boolean*
      - This determines whether other users can use your StackScript.
    * - :code:`--label`
      - *string*
      - The StackScript's label is for display purposes only.
    * - :code:`--mine`
      - *boolean*
      - Returns :code:`true` if this StackScript is owned by the account of the user making the request, and the user making the request is unrestricted or has access to this StackScript.
    * - :code:`--rev_note`
      - *string*
      - This field allows you to add notes for the set of revisions made to this StackScript.


------------

.. _commands_stackscripts_view:

`view <https://techdocs.akamai.com/linode-api/reference/get-stack-script>`_
---------------------------------------------------------------------------


Returns all of the information about a specified StackScript, including the contents of the script.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli stackscripts view [-h]
    	[--deployments_total deployments_total] [--label label]
    	[--description description] [--mine mine]
    	[--is_public is_public] [--rev_note rev_note]
    	[--order-by {deployments_total,label,description,mine,is_public,rev_note}]
    	[--order {asc,desc}] stackscriptId


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli stackscripts view 10079

.. _commands_stackscripts_create:

`create <https://techdocs.akamai.com/linode-api/reference/post-add-stack-script>`_
----------------------------------------------------------------------------------


Creates a StackScript in your Account.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli stackscripts create [-h] [--description description]
    	[--label label] [--script script] [--images images]
    	[--rev_note rev_note]
    	[--user_defined_fields user_defined_fields]
    	[--is_public is_public]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli stackscripts create \
    	--label a-stackscript \
    	--description "This StackScript install and configures MySQL" \
    	--images "linode/debian9" \
    	--images "linode/debian8" \
    	--is_public true \
    	--rev_note "Set up MySQL" \
    	--script '#!/bin/bash'


.. cssclass:: action-subheading

    .. rubric:: Arguments

.. cssclass:: action-subheading-description

    Additional fields used to execute this request.

      - No
      - integer
      - The total number of times this StackScript has been deployed.
    * - :code:`--description`
      - No
      - string
      - A description for the StackScript.
    * - :code:`--id`
      - No
      - integer
      - The unique ID of this StackScript.
    * - :code:`--is_public`
      - No
      - boolean
      - This determines whether other users can use your StackScript.
    * - :code:`--mine`
      - No
      - boolean
      - Returns :code:`true` if this StackScript is owned by the account of the user making the request, and the user making the request is unrestricted or has access to this StackScript.
    * - :code:`--rev_note`
      - No
      - string
      - This field allows you to add notes for the set of revisions made to this StackScript.
    * - :code:`--updated`
      - No
      - string
      - The date this StackScript was last updated.

        :ref:`(section) <commands_stackscripts_create_argument_sections_user_defined_fields>`
      - No
      - json
      - A custom field defined by the User with a special syntax within a StackScript.
    * - :code:`--user_gravatar_id`
      - No
      - string
      - The Gravatar ID for the User who created the StackScript.
    * - :code:`--username`
      - No
      - string
