.. _commands_networking_ip-view:

`ip-view <https://techdocs.akamai.com/linode-api/reference/get-ip>`_
--------------------------------------------------------------------


Returns information about a single IP Address on your Account.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli networking ip-view [-h] [--order-by {}]
    	[--order {asc,desc}] address


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

.. _commands_networking_ip-add:

`ip-add <https://techdocs.akamai.com/linode-api/reference/post-allocate-ip>`_
-----------------------------------------------------------------------------


Allocates a new IPv4 Address on your Account.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli networking ip-add [-h] [--type type] [--public public]
    	[--linode_id linode_id]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli networking ip-add \
    	--type ipv4 \
    	--public true \
    	--linode_id 123


.. cssclass:: action-subheading

    .. rubric:: Arguments

.. cssclass:: action-subheading-description

    Additional fields used to execute this request.


.. rst-class:: action-argument-section-table

..  list-table::
    :header-rows: 1
    :width: 100%
    :widths: 1 10 10 79

    * - Name
      - Required
      - Type
      - Description

    * - :code:`--linode_id`
      - :action-argument-required:`Yes`
      - integer
      - The ID of a Linode you have access to that this address will be allocated to.
    * - :code:`--public`
      - :action-argument-required:`Yes`
      - boolean
      - Whether to create a public or private IPv4 address.
    * - :code:`--type`
      - :action-argument-required:`Yes`
      - string
      - The type of address you are requesting.

.. _commands_networking_ips-list:

`ips-list <https://techdocs.akamai.com/linode-api/reference/get-ips>`_
----------------------------------------------------------------------


Returns a paginated list of IP addresses on your account, excluding private addresses.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli networking ips-list [-h] [--address address]
    	[--region region] [--rdns rdns] [--prefix prefix] [--type type]
    	[--order-by {address,region,rdns,prefix,type}]
    	[--order {asc,desc}]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli networking ips-list


.. cssclass:: action-subheading

    .. rubric:: Filterable Attributes


.. cssclass:: action-subheading-description

    Arguments used to define a filter for response entries.


.. rst-class:: action-filterable-field-table

..  list-table::
    :header-rows: 1
    :width: 100%
    :widths: 15 10 75

    * - Name
      - Type
      - Description

    * - :code:`--address`
      - *string*
      - The IP address.
    * - :code:`--prefix`
      - *integer*
      - The number of bits set in the subnet mask.
    * - :code:`--rdns`
      - *string*
      - The reverse DNS assigned to this address.
    * - :code:`--region`
      - *string*
      - The Region this IP address resides in.
    * - :code:`--type`
      - *string*
      - The type of address this is.
