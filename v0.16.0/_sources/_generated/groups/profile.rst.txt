Profile

This section details Profile-related Linode CLI commands.

`view <https://techdocs.akamai.com/linode-api/reference/get-profile>`_

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli profile view [-h] [--order-by {}] [--order {asc,desc}]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli profile view

.. _commands_profile_update:

`update <https://techdocs.akamai.com/linode-api/reference/put-profile>`_
------------------------------------------------------------------------


Update information in your Profile.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli profile update [-h]
    	[--two_factor_auth two_factor_auth]
    	[--authorized_keys authorized_keys]
    	[--ip_whitelist_enabled ip_whitelist_enabled] [--email email]
    	[--email_notifications email_notifications]
    	[--restricted restricted] [--timezone timezone]
    	[--lish_auth_method lish_auth_method]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli profile update \
    	--email example-user@gmail.com \
    	--timezone US/Eastern \
    	--email_notifications true \
    	--list_auth_method keys_only \
    	--two_factor_auth true \
    	--restricted false


.. cssclass:: action-subheading

    .. rubric:: Arguments

.. cssclass:: action-subheading-description

    Additional fields used to execute this request.


.. rst-class:: action-argument-section-table

..  list-table::
    :header-rows: 1
    :width: 100%
    :widths: 1 10 10 79

    * - Name
      - Required
      - Type
      - Description

    * - :code:`--authentication_type`
      - No
      - string
      - This account's Cloud Manager authentication type.
    * - :code:`--authorized_keys`
      - No
      - array[string]
      - The list of SSH Keys authorized to use Lish for your User.
    * - :code:`--email`
      - No
      - string
      - Your email address.
    * - :code:`--email_notifications`
      - No
      - boolean
      - If true, you will receive email notifications about account activity.
    * - :code:`--ip_whitelist_enabled`
      - No
      - boolean
      - If true, logins for your User will only be allowed from whitelisted IPs.
    * - :code:`--lish_auth_method`
      - No
      - string
      - The authentication methods that are allowed when connecting to `the Linode Shell (Lish) <https://www.linode.com/docs/guides/lish/>`_.
    * - :code:`--referrals`
      - No
      - object
      - Information about your status in our referral program.
    * - :code:`--restricted`
      - No
      - boolean
      - If true, your User has restrictions on what can be accessed on your Account.
    * - :code:`--timezone`
      - No
      - string
      - The timezone you prefer to see times in.
    * - :code:`--two_factor_auth`
      - No
      - boolean
      - If true, logins from untrusted computers will require Two Factor Authentication.
    * - :code:`--uid`
      - No
      - integer
      - Your unique ID in our system.
    * - :code:`--username`
      - No
      - string
      - Your username, used for logging in to our system.
    * - :code:`--verified_phone_number`
      - No
      - string
      - The phone number verified for this Profile with the `Verify a phone number <https://techdocs.akamai.com/linode-api/reference/post-profile-phone-number-verify>`_ operation.
