VLANs
=====

This section details VLAN-related Linode CLI commands.

.. _vlans_list:

`list <https://techdocs.akamai.com/linode-api/reference/get-vlans>`_
--------------------------------------------------------------------


Returns a list of all Virtual Local Area Networks (VLANs) on your Account.

.. cssclass:: action-subheading

    .. rubric:: Usage

:action-subheading-description:`The format accepted by this command.`

..  code-block:: bash

    linode-cli vlans list [-h] [--region region] [--label label]
    	[--order-by {region,label}] [--order {asc,desc}]


.. cssclass:: action-subheading

    .. rubric:: Sample

:action-subheading-description:`Examples of how this command might be used.`


..  code-block:: bash

    linode-cli vlans list


.. rubric:: Filterable Attributes

`Arguments used to define a filter for response entries.`

..  list-table::
    :header-rows: 1
    :width: 100%
    :widths: 15 10 75

    * - Name
      - Type
      - Description

    * - :code:`--label`
      - *string*
      - The name of this VLAN.
    * - :code:`--region`
      - *string*
      - This VLAN's data center region.
