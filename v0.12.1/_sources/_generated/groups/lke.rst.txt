.. _lke_cluster-create:

`cluster-create <https://techdocs.akamai.com/linode-api/reference/post-lke-cluster>`_
-------------------------------------------------------------------------------------


Creates a Kubernetes cluster.

.. cssclass:: action-subheading

    .. rubric:: Usage

:action-subheading-description:`The format accepted by this command.`

..  code-block:: bash

    linode-cli lke cluster-create [-h] [--tags tags]
    	[--region region] [--label label]
    	[--control_plane.high_availability high_availability]
    	[--k8s_version k8s_version] [--node_pools node_pools]
    	[--node_pools.taints taints] [--node_pools.disks disks]
    	[--node_pools.count count] [--node_pools.tags tags]
    	[--node_pools.type type] [--node_pools.labels.value value]
    	[--node_pools.labels.key key]
    	[--node_pools.autoscaler.enabled enabled]
    	[--node_pools.autoscaler.max max]
    	[--node_pools.autoscaler.min min]


.. cssclass:: action-subheading

    .. rubric:: Sample

:action-subheading-description:`Examples of how this command might be used.`


..  code-block:: bash

    linode-cli lke cluster-create \
    	--label cluster12345 \
    	--region us-central \
    	--k8s_version 1.27 \
    	--control_plane.high_availability true \
    	--node_pools.type g6-standard-4 --node_pools.count 6 \
    	--node_pools.type g6-standard-8 --node_pools.count 3 \
    	--node_pools.autoscaler.enabled true \
    	--node_pools.autoscaler.max 12 \
    	--node_pools.autoscaler.min 3 \
    	--tags ecomm


.. cssclass:: action-subheading

    .. rubric:: Arguments

:action-subheading-description:`Additional fields used to execute this request.`

..  list-table::
    :header-rows: 1
    :width: 100%
    :widths: 15 10 10 65

    * - Name
      - Required
      - Type
      - Description

    * - :code:`--k8s_version`
      - :action-argument-required:`Yes`
      - string
      - The desired Kubernetes version for this Kubernetes cluster in the format of <major>.<minor>, and the latest supported patch version will be deployed.
    * - :code:`--label`
      - :action-argument-required:`Yes`
      - string
      - This Kubernetes cluster's unique label for display purposes only.
    * - :code:`--node_pools.count`
      - :action-argument-required:`Yes`
      - integer
      - The number of nodes in the Node Pool.
    * - :code:`--node_pools.type`
      - :action-argument-required:`Yes`
      - string
      - The Linode Type for all of the nodes in the Node Pool.
    * - :code:`--region`
      - :action-argument-required:`Yes`
      - string
      - This Kubernetes cluster's location.
    * - :code:`--control_plane.high_availability`
      - No
      - boolean
      - Defines whether High Availability is enabled for the Control Plane Components of the cluster.
    * - :code:`--node_pools`
      - No
      - object
      - Specifies a collection of Linodes which will be members of a Kubernetes cluster.
    * - :code:`--node_pools.autoscaler.enabled`
      - No
      - boolean
      - Whether autoscaling is enabled for this Node Pool.
    * - :code:`--node_pools.autoscaler.max`
      - No
      - integer
      - The maximum number of nodes to autoscale to.
    * - :code:`--node_pools.autoscaler.min`
      - No
      - integer
      - The minimum number of nodes to autoscale to.
    * - :code:`--node_pools.disks`
      - No
      - object
      - This Node Pool's custom disk layout.
    * - :code:`--node_pools.labels.key`
      - No
      - string
      - The Kubernetes label key.
    * - :code:`--node_pools.labels.value`
      - No
      - string
      - The Kubernetes label value.
    * - :code:`--node_pools.tags`
      - No
      - object
      - An array of tags applied to this object.
    * - :code:`--node_pools.taints`
      - No
      - object
      - Kubernetes taints to add to node pool nodes.
    * - :code:`--tags`
      - No
      - array[string]
      - An array of tags applied to the Kubernetes cluster.
