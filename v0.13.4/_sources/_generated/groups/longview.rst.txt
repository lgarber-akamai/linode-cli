Longview

.. _commands_longview_list:

`list <https://techdocs.akamai.com/linode-api/reference/get-longview-clients>`_
-------------------------------------------------------------------------------


Returns a paginated list of Longview Clients you have access to.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli longview list [-h] [--label label]
    	[--order-by {label}] [--order {asc,desc}]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.

    linode-cli longview list

.. _commands_longview_create:

`create <https://techdocs.akamai.com/linode-api/reference/post-longview-client>`_
---------------------------------------------------------------------------------


Creates a Longview Client.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli longview create [-h] [--label label]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli longview create \
    	--label client789


.. cssclass:: action-subheading

    .. rubric:: Arguments

.. cssclass:: action-subheading-description

    Additional fields used to execute this request.


.. rst-class:: action-argument-section-table

..  list-table::
    :header-rows: 1
    :width: 100%
    :widths: 1 10 10 79

    * - Name
      - Required
      - Type
      - Description

    * - :code:`--api_key`
      - No
      - string
      - The API key for this Client, used when configuring the Longview Client application on your Linode.
    * - :code:`--apps`
      - No
      - object
      - The apps this Client is monitoring on your Linode.
    * - :code:`--created`
      - No
      - string
      - When this Longview Client was created.
    * - :code:`--id`
      - No
      - integer
      - This Client's unique ID.
    * - :code:`--install_code`
      - No
      - string
      - The install code for this Client, used when configuring the Longview Client application on your Linode.
    * - :code:`--label`
      - No
      - string
      - This Client's unique label.
    * - :code:`--updated`
      - No
      - string
      - When this Longview Client was last updated.
