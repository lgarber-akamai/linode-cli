Volumes
=======

This section details Volume-related Linode CLI commands.

.. _commands_volumes_list:

`list <https://techdocs.akamai.com/linode-api/reference/get-volumes>`_
----------------------------------------------------------------------


Returns a paginated list of Volumes you have permission to view.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli volumes list [-h] [--label label] [--tags [tags] ]
    	[--order-by {label,tags}] [--order {asc,desc}]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli volumes list

    * - :code:`--label`
      - *string*
      - The Volume's label is for display purposes only.
    * - :code:`--tags`

.. _commands_volumes_create:

`create <https://techdocs.akamai.com/linode-api/reference/post-volume>`_
------------------------------------------------------------------------


Creates a Volume on your Account.

.. cssclass:: action-subheading

    .. rubric:: Usage


.. cssclass:: action-subheading-description

    The format accepted by this command.

..  code-block:: bash

    linode-cli volumes create [-h] [--config_id config_id]
    	[--linode_id linode_id] [--region region] [--size size]
    	[--tags tags] [--label label]


.. cssclass:: action-subheading

    .. rubric:: Sample

.. cssclass:: action-subheading-description

    Examples of how this command might be used.


..  code-block:: bash

    linode-cli volumes create \
    	--label my-volume \
    	--size 20 \
    	--linode_id 12346 \
    	--no-defaults


.. cssclass:: action-subheading

    .. rubric:: Arguments

.. cssclass:: action-subheading-description

    Additional fields used to execute this request.


.. rst-class:: action-argument-section-table

..  list-table::
    :header-rows: 1
    :width: 100%
    :widths: 1 10 10 79

    * - Name
      - Required
      - Type
      - Description

    * - :code:`--label`
      - :action-argument-required:`Yes`
      - string
      - The Volume's label, which is also used in the :code:`filesystem_path` of the resulting volume.
    * - :code:`--config_id`
      - No
      - integer
      - When creating a Volume attached to a Linode, the ID of the Linode Config to include the new Volume in.
    * - :code:`--linode_id`
      - No
      - integer
      - The Linode this volume should be attached to upon creation.
    * - :code:`--region`
      - No
      - string
      - The Region to deploy this Volume in.
    * - :code:`--size`
      - No
      - integer
      - The initial size of this volume, in GB.
    * - :code:`--tags`
      - No
      - array[string]
      - An array of Tags applied to this object.

.. rst-class:: action-argument-section-table

..  list-table::
    :header-rows: 1
    :width: 100%
    :widths: 1 10 10 79

    * - Name
      - Required
      - Type
      - Description

    * - :code:`--created`
      - No
      - string
      - When this Volume was created.
    * - :code:`--filesystem_path`
      - No

      - No
      - string
      - The Volume's label is for display purposes only.
    * - :code:`--linode_id`
      - No
      - integer
      - If a Volume is attached to a specific Linode, the ID of that Linode will be displayed here.
    * - :code:`--linode_label`
      - No
      - string
      - If a Volume is attached to a specific Linode, the label of that Linode will be displayed here.
    * - :code:`--region`
      - No
      - string
      - The unique ID of this Region.
    * - :code:`--size`
      - No
      - integer
      - The Volume's size, in GiB.
    * - :code:`--status`
      - No
      - string
      - The current status of the volume.
    * - :code:`--tags`
      - No
      - array[string]
      - An array of Tags applied to this object.
    * - :code:`--updated`
      - No
      - string
      - When this Volume was last updated.
